## 使用说明

- 这是发给 Claude Code 的第一条消息
- 如果 Claude Code 没按要求先读文档，就直接让它回到这条消息重新执行
- 如果 Claude Code 开始擅自扩需求，也直接引用这条消息和 `docs/claude-code-执行说明.md`

## 发送后的预期

Claude Code 的第一轮回复，应该至少包含：

- 对一期目标的复述
- 对首个任务的选择
- 对文档阅读顺序的确认
- 对 `docs/任务清单.md` 更新方式的确认

如果第一轮回复里直接开始写代码，说明它没有严格按交接流程执行，应要求它回退。
