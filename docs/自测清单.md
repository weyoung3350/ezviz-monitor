# 自测清单

## 1. 用途

本清单用于约束 Claude Code 在交付 HA 一期实现前，至少完成与当前范围匹配的验证。

## 2. 基本要求

- 每项验证都要记录实际方法
- 若无法执行，要明确写出原因
- 自动化校验优先于口头说明
- 无法在当前仓库直接运行的 HA 验证，可用 trace、开发者工具、实际服务调用记录代替
- 验证结果需标明由哪个角色完成

## 3. 必做检查

- 童锁状态实体
- 童锁开启服务
- 摄像头快照方式
- Telegram 发送方式
- AppDaemon `NotifyService` 加载状态与监听方式
