# 自测清单

## 1. 用途

本清单用于约束 Claude Code 在交付 HA 一期实现前，至少完成与当前范围匹配的验证。

## 2. 基本要求

- 每项验证都要记录实际方法
- 若无法执行，要明确写出原因
- 自动化校验优先于口头说明
- 无法在当前仓库直接运行的 HA 验证，可用 trace、开发者工具、实际服务调用记录代替
- 验证结果需标明由哪个角色完成

## 3. 必做检查

改动要点：
- `on_notify_request` 改用 AppDaemon 原生 `self.run_in_executor(partial(...))` 替代 `asyncio.to_thread`，任务归属随 app reload 正确清理

- `_send_dingtalk` 校验 `image_path` 前缀，不合法即降级为纯文字
- `apps.yaml.example` 增加内网 URL 限制的醒目注释
