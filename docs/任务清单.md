## 2. 状态定义

- `未开始`：尚未动手
- `进行中`：正在实现或调试
- `待Review`：代码和自测已完成，等待 codex review
- `需返工`：review 后发现问题，需要继续修改
- `已完成`：review 通过或确认当前阶段完成
- `已阻塞`：因环境、依赖或需求问题无法继续

## 3. 更新规则

Claude Code 每次更新任务时，至少要填写：

- 状态
- 更新日期
- 修改文件
- 自测命令
- 自测结果
- 风险/阻塞

codex review 时会补：

- Review 结论
- Review 备注
