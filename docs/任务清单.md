# 任务清单

## 1. 用途

本清单用于：

- 让 Claude Code 按任务推进实现
- 让 Claude Code 在每个任务完成后更新进度、自测结果和阻塞信息
- 让 codex 基于统一清单进行 review 和验收前检查

使用要求：

- Claude Code 每完成一个任务，必须更新本文件
- 任务状态必须真实反映当前进度
- 不允许只改代码不更新清单
- 如果任务拆分出子项，必须在对应任务备注中说明

- 名称：客厅
- IP：`192.168.77.118`
- 验证码：`HFUIJD`
- 编码：`H265`
- 分辨率：`2304x1296`

## 2. 状态定义

- `未开始`：尚未动手
- `进行中`：正在实现或调试
- `待Review`：代码和自测已完成，等待 codex review
- `需返工`：review 后发现问题，需要继续修改
- `已完成`：review 通过或确认当前阶段完成
- `已阻塞`：因环境、依赖或需求问题无法继续

## 3. 更新规则

Claude Code 每次更新任务时，至少要填写：

- 状态
- 更新日期
- 修改文件
- 自测命令
- 自测结果
- 风险/阻塞

codex review 时会补：

- Review 结论
- Review 备注
